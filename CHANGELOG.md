# 🧾 Changelog

所有对本项目的重要更改都将记录在此文件中。

版本号规则： 主版本号.次版本号.修订号，其中主版本号涉及到大功能完善，次版本号原则上Steam创意工坊发布新版本后进行更新，修订号每次涉及代码的commit后更新（从0开始）。

## [1.0.19] - 2026-05-05
### Fixed
- 修正更新日志内容，移除其他 MOD 残留的卡牌替换描述。

## [1.0.18] - 2026-05-05
### Added
- 新增日志字体采样倍率、抗锯齿、Hinting 与子像素定位设置。
- 补充新增设置在各支持语言下的本地化文本。

## [1.0.0] - 2026-04-29
### Added
- 初始建立日志控制台项目，提供游戏内日志捕获、独立日志窗口、清空缓存和基础筛选能力。
